# 📝 daryan notes.

A modern, minimal, and responsive Notes App built with **HTML**, **CSS**, and **Vanilla JavaScript**. Easily create, edit, delete, and manage your notes with automatic local storage support.

## ✨ Features

* 📝 Create new notes
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 💾 Save notes using Local Storage
* 🌙 Dark / Light mode
* 📱 Fully responsive design
* 🍔 Mobile hamburger menu
* 🔔 Animated notification alerts
* ⚡ Instant UI updates without page reload

## 📂 Features Overview

### Create Notes

Create a new note by entering a title. New notes are automatically initialized with default content and stored in Local Storage.

### Edit Notes

Open a note in the editor, modify the title or content, then save the changes instantly.

### Delete Notes

Remove notes permanently with a single click.

### Local Storage

All notes are saved locally in your browser, so they remain available even after refreshing or closing the page.

### Dark Mode

Switch between Light and Dark themes. Your preferred theme is remembered using Local Storage.

### Responsive Design

Optimized for desktop, tablet, and mobile devices.

### Notification Alerts

Animated notifications appear whenever you:

* Add a note
* Save changes
* Delete a note
